# javascript를 이용하여 제작한 그림판 클론코딩
******
###### 작업기간 : 0721 ~ 0721
###### 사용언어 : javascript, html, css
******
###### painting.js으로 설계된 그림판 기능 목록
* 색을 설정하여 라인을 따라 그리는 기능.
* 라인의 크기를 range값을 부여해 조절하는 기능.
* canvas화면을 클릭 시 적용되고 있던 색으로 fill하는 기능.
* erase버튼을 통해 canvas의 그림을 지우는 기능.
* 완성한 이미지를 저장하는 기능.
